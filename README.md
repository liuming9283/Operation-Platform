## 项目概述

打算搭建一个简单的运维平台，平台采用python的Django框架实现。采用的Django版本为1.9.5，python解释器版本为2.7.11，本项目主要作为本人学习python和Django使用，目前只是搭建了几个简单的web页面与一些简单的表，同时实现了一个简单的内部聊天室功能，其他的功能会在后续学习中进行扩充。
